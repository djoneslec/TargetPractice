# Target Practice
Building a time-trial FPS level in Unity game engine.

This workshop builds on a short session discussing how ‘raycast’ shooting works in games development. The Unity version used in this project is 6000.0.58f2. It’s free to download and use unity, which is available from: https://unity.com/download
